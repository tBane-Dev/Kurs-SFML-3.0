# Introduction

![SFML](sfml-logo-big.png)

Welcome to this tutorial dedicated to SFML 3.0.

SFML (Simple and Fast Multimedia Library) is a C++ library that allows you to create multimedia applications, 2D games, and simple graphical tools. Thanks to its simple and intuitive interface, it lets you focus on learning programming and bringing your ideas to life instead of dealing with the complex configuration of low-level libraries.

The goal of this tutorial is to present the most important features of SFML 3.0 in a practical way. Rather than focusing solely on theory, we will build working examples that demonstrate how to use windows, graphics, textures, text, audio, events, and other capabilities provided by the library.

This tutorial assumes a basic knowledge of C++. If you can create functions and classes, use standard containers, and compile programs in your development environment, you should have no trouble following the examples presented here.
